## 「미·중 패권 경쟁 속 대한민국의 전략적 위치 분석」
— 실리외교 국가 비교를 통한 데이터 기반 시사점

---
### MOTIVATION

평소에 시도해보고 싶던 주제였기 때문에, 여러 요소들을 종합하여 한번 도전해보고자 한다. 비록 해당 분야의 전공자는 아니고 데이터 사이언스를 접한 지도 오래되지 않았지만, 완벽함보다는 첫 삽을 뜨고 이후에 발전시켜 나가자는 생각으로 본 작업을 진행하기로 결정했다.

물론 과정 전반에서 미흡한 부분이 많을 수 있고, 경우에 따라 결과가 기대에 미치지 못할 수도 있다. 그럼에도 불구하고 이번 작업은 내가 고민해왔던 생각들을 실제로 투영해 보고, 어떤 관점에서 문제를 설계하고 어떻게 접근했는지를 보여주는 데에 의미가 있다고 생각한다.  

그러한 맥락에서 전체적인 설계와 진행 과정을 중심으로 봐주시면 감사하겠습니다. 

성을 높였습니다.

## 📌 프로젝트 개요

본 프로젝트는 **미·중 패권 경쟁이라는 구조적 환경 속에서**  
대한민국의 **향후 전략적 발전 방향을 데이터 기반으로 분석**하는 것을 목표로 합니다.

❌ 단순한 외교·안보 담론  
❌ 포괄적이고 추상적인 정책 제언  

대신,  
✅ **정량 지표에 근거한 현실적 비용–편익 분석**  
✅ **산업·안보·공급망·기술을 포괄하는 전략 포트폴리오 접근**  
✅ **조건부·단계적·실행 가능한 시사점 도출**을 지향합니다.

## 🎯 핵심 연구 질문

> **미·중 패권 경쟁 환경에서  
대한민국은 어떠한 전략적 선택을 통해  
국가이익(성장·안정·자율성)을 최적화할 수 있는가?**

본 프로젝트의 모든 분석은 위 질문에 답하는 데 기여하도록 설계됩니다.

## 🧠 분석 철학 (Analysis Philosophy)

- 분석 대상은 “발전 여부”가 아니라 **“최적화”**
- 전략은 정치적 주장이나 당위가 아닌 **의사결정 문제(Decision Problem)**로 취급
- 모든 결론은 다음을 포함해야 함:
  - 📊 데이터
  - 📐 가정
  - 💰 비용과 편익
  - ⚠️ 리스크(특히 꼬리위험)

👉 **정량화할 수 없는 주장에는 신중**하게 접근합니다.
